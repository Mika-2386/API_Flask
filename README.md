# Это проект для работы с csv файлами

1. скачайте себе данный проект 
2. для работы можно воспользоваться существующим файлом salary_developer.CSV на него настроен проект.
3. Преходите в корень проекта устанавливаете venv, pip-tools 
4. переходите в (venv)
5. устанавливаете зависимости 
5. запускаете проек flask --app .\acme\server.py run
6. переходите по ссылке http://127.0.0.1:5000
7. можно подгрузить файл selery_developer (он выгружен из postgres)
8. можно сформировать из базы.
9. Запустить анализ выдаст ниже результат анализа таблицы
10. Нажать построить график - построит график на отдельной странице
11. И можно скачать файл. 
12. так же можно подключиться к своей БД 
12.1. задать в файле api app.config['SQLALCHEMY_DATABASE_URI'] = 'своои параметры базы'
13. указать свою модель в models.py
14. В def analyze_salary указать свою модель и на основе чего мы строим зависимости
15. так же указать  в def plot_salary и generate_report
